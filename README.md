# Employee Retention Prediction

## Business Problem
High employee attrition increases hiring costs and reduces team productivity.
This project uses Machine Learning to predict employees who are likely to leave.

## Dataset Overview
- Demographics
- Education & Experience
- Company details
- Training hours
- Target variable (Attrition)

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- LightGBM
- Matplotlib, Seaborn

## Models Used
- Logistic Regression
- Random Forest
- XGBoost
- LightGBM (Best Model)

## Best Model Performance
- Accuracy: ~79%
- ROC-AUC: ~81%
